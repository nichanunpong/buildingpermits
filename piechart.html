Distribution of Permits by Type of Work

Which type of work had the highest number of building permits issued?

How to Use This Chart click

1. Select a Year (from the filter on the right) to update the chart.
2. Hover over each slice of the pie chart to see the type of work and the number of permits.
3. Bigger slices represent more permits issued for that type of work.
4. Use the year filter to check how the distribution changes over different years.
5. Tip: Focus on the largest segments to understand the main types of construction activities happening.
