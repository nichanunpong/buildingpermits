Building Permits Dashboard

How to Use This Dashboard click

1. Filter by Year: Use the year input box to update all charts.
2. Summary Cards: Show Total Permits, Total Project Value, and Average Project Value, including % change from the previous year.
3. Click a Bar: In the Top 10 Neighborhoods chart, click a bar to filter all charts by that neighborhood.
4. Explore Trends:
    Line Chart shows permits issued month by month.
    Map shows where permits were issued (darker = more permits).
    Pie Chart shows types of work by permit distribution.
